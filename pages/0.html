In order to set you up for success, we need a few things in place before we can start:
1
Please login to https://demo.okta.com. You won't need to provide anything, this is our demo instance.
2
Make sure you have Node.js Version 16 or higher installed and a code editor ready. We will use Visual Studio Code for the live coding, but any other editor that you feel comfortable with should be good as well. Lastly make sure you have cloned the lab repository to your local machine.

Create a new directory where you can use this workwhop:
cd ~/ && mkdir okta-cic && cd okta-cic

Then us the following command should get you ready to go:
git clone https://github.com/escobar022/okta-cic-workshop-sandbox.git && cd okta-cic-workshop-sandbox && npm i
Once you are set up, feel free to start with the first lab around integrating Auth0 into your application. To the next lab
